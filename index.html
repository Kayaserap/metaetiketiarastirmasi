META ETİKEYİ NEDİR, NE İŞE YARAR?

Meta etiketleri bir web sayfasının içeriğini tanımlayan HTML etiketleridir.Sayfanın başlık ,karakter seti, açıklama gibi meta bilgilerini içerir. Meta etiketleri, kullanıcıların web sitesini rahatlıkla bulmasını sağlayan SEO avantajı yaratır. Google dahil birçok arama motoru tarafından desteklenmektedir. Meta etiketler HTML sayfasının < head > bölümüne eklenir ve diğer HTML etiketlerinin aksine sayfada kullanıcılar tarafından görünemez ancak web sayfasının kaynak kodundan görüntülenebilirler.



    META ETİKETLERİ
    1. Site karekter içeriği: Charset özelliği HTML belgesinin karakter kodlamasını belirtir.Türkçe karakter seçeneği için UFT-8 kullanılır.
      < meta charset=”UTF-8 />
    2. Tüm site hakkında bilgi veren cümledir .
      < meta name="Abstract" content="Site özeti" >
    3. Sitenin telif hakları ile ilgili bilgi verir.
      < name="Copyright" content="telif hakkı cümlesi" >
    4. Site sahibi,yazarı ile ilgili bilgi verir.
      < meta name="author" content="" />
    5. Sitenizin dilini belirler.
      < meta http-equiv="content-language" content="tr">
    6. Site açıklaması için kullanılır.a Site açıklması 160 karakteri geçmeyecek kadar kısa olmalı.
      < meta name="description" content="" />
    7. siteye gelen robotların nasıl davranacağını belirler.
      < meta name="robots" content="index,follow" />
